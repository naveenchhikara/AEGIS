## v1 Requirements (Clickable Prototype Milestone)

### Authentication & Access

- [x] **AUTH-01**: User sees login screen with logo, email/password fields, and MFA prompt UI
- [x] **AUTH-02**: User can select language (EN/HI/MR/GU) from login screen dropdown
- [x] **AUTH-03**: Demo login accepts any valid email format and redirects to dashboard (no real auth)

### Navigation & Layout

- [x] **NAV-01**: Sidebar navigation shows all 7 main screens
- [x] **NAV-02**: Top bar shows user profile dropdown, language switcher, notifications bell
- [x] **NAV-03**: Layout is responsive — sidebar collapses to hamburger menu on mobile
- [x] **NAV-04**: All screens load without page refresh (client-side routing)

### Dashboard

- [x] **DASH-01**: Compliance health score displayed (0-100) with color indicator (red/yellow/green)
- [x] **DASH-02**: Donut chart showing audit coverage (audited entities / total entities)
- [x] **DASH-03**: Count cards: Total findings, Critical findings, Open findings, Overdue findings
- [x] **DASH-04**: Risk indicator panel (Overall risk level with contributing factors)
- [x] **DASH-05**: Regulatory calendar widget showing upcoming compliance deadlines
- [x] **DASH-06**: Quick action buttons: New Finding, New Compliance Task, View Audit Plan

### Compliance Registry

- [x] **COMP-01**: Filterable table showing 50+ RBI compliance requirements
- [x] **COMP-02**: Sortable columns: Requirement ID, Category, Description, Status, Due Date, Evidence Count, Assigned To
- [x] **COMP-03**: Status badges: Compliant (green), Partial (yellow), Non-Compliant (red), Pending (gray)
- [x] **COMP-04**: Category filter dropdown: Risk Management, Governance, Operations, IT, Credit, Market Risk
- [x] **COMP-05**: Click on requirement row opens detail modal with full description and evidence list
- [x] **COMP-06**: Compliance health trend chart (last 6 months)

### Audit Planning

- [x] **AUDT-01**: Annual plan calendar view showing planned audits by month
- [x] **AUDT-02**: Engagement cards showing: Audit name, branch/department, planned dates, assigned team, status
- [x] **AUDT-03**: Status indicators: Not Started (gray), In Progress (blue), On Hold (orange), Complete (green)
- [x] **AUDT-04**: Filter by audit type: Branch Audit, IS Audit, Credit Audit, Compliance Audit, Revenue Audit
- [x] **AUDT-05**: Progress bar for each engagement showing completion percentage
- [x] **AUDT-06**: Click on engagement opens workspace with audit program linkages

### Finding Management

- [x] **FIND-01**: Findings list table with severity badges: Critical (red), High (orange), Medium (yellow), Low (green)
- [x] **FIND-02**: Filterable columns: Finding ID, Title, Category, Severity, Status, Assigned Auditor, Age (days)
- [x] **FIND-03**: Status dropdown filter: Draft, Under Review, Response Pending, Action Planned, Closed
- [x] **FIND-04**: Click on finding opens detail page with all fields and timeline
- [x] **FIND-05**: Finding detail shows: Observation, Root Cause, Risk Impact, Auditee Response, Action Plan, Evidence attachments
- [x] **FIND-06**: Timeline view showing status change history with dates and actors

### Board Report

- [x] **RPT-01**: Executive summary section with key metrics and risk summary
- [x] **RPT-02**: Audit coverage table: Planned vs Actual audits by category
- [x] **RPT-03**: Key findings summary: Top 10 critical findings with brief descriptions
- [x] **RPT-04**: Compliance scorecard: Overall score plus category breakdowns
- [x] **RPT-05**: Recommendations section with prioritized action items
- [x] **RPT-06**: Print/PDF preview mode with clean formatting

### RBI Circulars & Common Observations

- [x] **RBI-01**: Compliance requirements sourced from provided RBI circular PDFs
- [x] **RBI-02**: Common audit observations populated from RBI examination patterns
- [x] **RBI-03**: Findings dataset includes realistic RBI-style observations

### Multi-Language

- [x] **I18N-01**: All UI labels translated to Hindi, Marathi, Gujarati
- [x] **I18N-02**: Language switcher in top bar persists preference
- [x] **I18N-03**: Banking terminology correctly translated (verified by domain expert)

### Demo Data

- [x] **DATA-01**: Apex Sahakari Bank profile: bank name, branch list, departments, staff directory
- [x] **DATA-02**: 50+ RBI compliance requirements with categories, due dates, statuses
- [x] **DATA-03**: 8-10 planned audits with team assignments and progress
- [x] **DATA-04**: 35 open findings with realistic descriptions, severities, root causes, responses
- [x] **DATA-05**: Compliance calendar with upcoming deadlines
